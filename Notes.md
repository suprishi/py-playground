#### There are two versions of Python out there:
- Python 2: Legacy version of Python and is going to be supported until year 2020
- Python 3: Python for the future

#### Python implementation
It is a program that understands Python syntax and can execute Python code (like a JavaScript engine).
- CPython: default implementation of Python (written in C). New features of Python, when introduced, are first added to CPython, and then gradually rolls out in the others below
- Jython: written in Java. Allows you to reuse some Java code in a Python program
- IronPython: written in C#. Allows you to reuse some C# code in a Python program
- PyPy: written in subset of Python
